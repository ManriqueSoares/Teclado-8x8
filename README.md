# Código para testar o teclado 8x8

A conexão está funcionando para o teclado invertido, preciso alterar a ordem entre os itens R's e os itens C's:

- R4 ------ GPIO 23
- R3 ------ GPIO 22
- R2 ------ GPIO 21
- R1 ------ GPIO 19

- C4 ------ GPIO 12
- C3 ------ GPIO 13
- C2 ------ GPIO 05
- C1 ------ GPIO 18
